What It Does

Core Features

🧠

Intent Recognition

Classifies user queries into intents using TF-IDF vectorization and Naive Bayes / Logistic Regression models trained on college data.

Python · scikit-learn
✂️

NLP Preprocessing

Tokenization, stopword removal, lemmatization and stemming using NLTK and spaCy pipelines for clean input processing.

NLTK · spaCy
🌐

REST API Backend

Python Flask server exposing clean API endpoints that the frontend calls to get real-time intelligent responses.

Flask · JSON API
💬

Context Awareness

Multi-turn conversation memory that tracks context across messages, so follow-up questions are answered intelligently.

Session Memory
📊

Admin Dashboard

Visualise chat analytics, most asked queries, response accuracy charts and user engagement statistics in real time.

Chart.js · Python
🗄️

Knowledge Database

SQLite database storing college FAQs, course info, timetables and department data — easily extensible for any institution.

SQLite · SQLAlchemy
